# FIPS Wind Tunnel v0.1.0

This first public contract release provides a deterministic Campaign v1alpha1
toolchain, individual and cohort/hybrid engines, causal accounting, search and
shrinking, pinned-daemon differential validation, immutable artifact analysis,
and the ten-family qualification atlas.

Headline boundaries:

- FIPS is pinned to `80c956a6fdb85dde1450969a21891c1158e43267`.
- TreeAnnounce is `168 + 32d` established-FMP bytes at that pin; depth 35 is
  1,288 bytes before lower-layer overhead.
- One-billion-node results are cohort/hybrid projections with uncertainty, not
  individual allocation.
- Daemon agreement is limited to observable telemetry; unsupported observations
  remain explicit.
- macOS and Linux are supported; Windows and real BLE/Tor/Nym calibration are
  outside v0.1.

Release evidence includes the qualification atlas, release audit, measured
benchmark, SPDX SBOM, SHA-256 checksums, package manifest, and platform
attestation. See `docs/quickstart.md`, `docs/support-matrix.md`, and
`docs/threat-model.md`.
